## Maths

Equations can be set easily. For example

$$ a^2 + b^2 = c^2 $$

can be set like this:

```
$$ a^2 + b^2 = c^2 $$
```

Inline maths is done with single-dollar quoted latex `$\forall_{i\in I}\,\alpha_i > 0$` becomes $\forall_{i\in I}\,\alpha_i > 0$.
